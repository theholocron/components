# Display Components

A series of components to display data.

## Installation

```bash
npm install --save-dev @theholocron/components-display
```

## Usage

```typescript
import { Component, type ComponentProps } from "@theholocron/components-display";

function App (props: ComponentProps) {
	return (
		<Component {...props} />
	);
}
```
